### Installing

Step 1. Clone the repo
```
git clone https://github.com/maxsmith009/NodeJsLab03.git
```

To access SQL application use 
```
cd SQL Part
```

To access ORM application use 
```
cd ORM Part
```

Step 2. Install dependencies
```
npm install
```

Step 3. Run server
```
npm start
```
